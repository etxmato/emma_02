Memory Type

The memory type of the main 64K CPU memory will be shown as soon as memory 'Type' (yellow ellipse) is selected. One field defines one page, i.e. 256 bytes of memory, so the top left field (blue ellipse) defines address 0000 to 00FF. To change the memory type of a page select the value and change it. Note that changing any of these values will likely crash the emulated computer; so use with care.

If memory type is shown in colour text the type or address range is linked to a hardware mapper, like COMX slots or banks and Elf EMS or ROM pagers. Note that the colour used is the same as the mapper selection features (orange and magenta ellipses)
