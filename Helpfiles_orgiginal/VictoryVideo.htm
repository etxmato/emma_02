Video Mode

Video Mode allows selection of PAL or NTSC version of the Studio III. The PAL version was designed for the European PAL TV system where the NTSC was used for the U.S. TV system. The PAL system has 192 visibly lines whereas the NTSC system has 128.

The NTSC system is now known as the Studio III which was never release by RCA. The PAL version was released as the Victory MPT-02.
