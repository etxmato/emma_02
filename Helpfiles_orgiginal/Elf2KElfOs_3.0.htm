Installation Elf/Os 3.0

Elf Os is a disk based operating system for Elf class computers designed by Mike Riley, for additional information see also: Elf/1802 Emulation.

For Elf/Os installation:

  • Start an Elf 2000 using ROM Elf2Kv110.bin (or later) and a VT simulator (serial or UART)
  • Select 'HEX' mode (instead of XMODEM) via the toggle button
  • Select file elfos_ram.hex via the HEX button
  • Unselect 'Auto'
  • Press play
  • When file transfer is done press return
  • Now type run 3000 (and return to set baud rate)
  • Execute step 1 to 5
  • Reset the Elf 2000
  • Select file supplement.hex via the HEX button
  • Press play
  • When file transfer is done press return
  • Now type run 3000 (and return to set baud rate)
  • Execute step 4 to install xrb.bin
  • Boot in ElfOs by typing boot or b

When using Elf/Os in combination with Intel-8275 video, please note the following:

  • I suggest to perform the installation on a serial or UART installation
  • The LOAD command will not work due to the fact that it uses the same memory location as the video memory. To solve this a LOAD command is included which is located at address hex 7400-74AE. This version is provided in file LOAD-8275.hex.
  • The @ command in the MINIMON will not work correctly due to the fact that it uses R0 as program counter which is not allowed when using Intel-8275 video. To solve this a MINIMON program is provided which uses R3 as program counter. This version is provided in file MINIMON-8275.hex.

The elf2k_installation.ide file includes a pre installed Elf/Os 3.2. The installation file will be renamed to elf2k.ide if the used file doesn't exist.
