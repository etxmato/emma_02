Installation Elf/Os 2.8

Elf Os is a disk based operating system for Elf class computers designed by Mike Riley, for additional information see also: Elf/1802 Emulation.

For Elf/Os installation on the Elf 2000 please follow the Elf 2000 manual available on the Spare Time Gizmos site: Elf 2000 User's Manual.

When using Elf/Os in combination with Intel-8275 video, please note the following:

  • For installation of ElfOs use elfos2k_0_2_8b.hex instead of elfos2k_0_2_8.hex.
  • The LOAD command will not work due to the fact that it uses the same memory location as the video memory. To solve this a LOAD command is included which is located at address hex 7400-74AE. This version is provided in file LOAD-8275.hex as well as stored on the elf2k_0_2_8.ide.
  • The @ command in the MINIMON will not work correctly due to the fact that it uses R0 as program counter which is not allowed when using Intel-8275 video. To solve this a MINIMON program is provided which uses R3 as program counter. This version is provided in file MINIMON-8275.hex as well as stored on the elf2k-installation.ide.

The elf2k_0_2_8.ide file includes a pre installed Elf/Os 2.8.
