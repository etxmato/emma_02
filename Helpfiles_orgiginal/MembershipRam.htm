Memory

To select the memory configuration select the memory choice button. The following configurations are supported:

  • 2 KB RAM
  • 4 KB RAM
  • 16 KB RAM
  • 32 KB RAM
  • 64 KB RAM
  • 32 KB ROM - 32 KB RAM
  • 32 KB RAM - 32 KB ROM

For the 32 KB ROM - 32 KB RAM configurations 32 KB ROM is configured from 0 to hex 7FFF and 32 KB RAM from hex 8000 to FFFF. For the 32 KB RAM - 32 KB ROM configurations 32 KB RAM is configured from 0 to hex 7FFF and 32 KB ROM from hex 8000 to FFFF.
