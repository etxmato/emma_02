ROM and RAM Files

The Cosmicos emulator can be started with a specific ROM file loaded, specified in the 'ROM' field. Default Cosmicos ROM location is the Cosmicos directory (located in the application data directory, see Directory and File Structure). To change the location use the 'ROM' button to browse for the file, do the same to change the filename or type in a new file name in the text field.

Note that ROM files in binary format, normally using extension .bin, will be loaded at c000. Intel hex and RCS Elf emulator hex format include storage locations and as such will be loaded in the specified address location.

RAM files can be loaded after starting the emulator by using the 'LOAD' button.
