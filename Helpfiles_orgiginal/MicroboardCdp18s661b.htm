CDP18S661B: Video (NTSC)

The CDP18S661B board includes:

  • RCA Video Interface System - CDP1869, CDP1870/CDP1876
  • NTSC, US standard colour video signal
  • Video display using 40 characters per line with 24 lines or 20 x 12 (double size) software selectable
  • On-board page and character memory
  • Character set expandable to 256 characters in RAM, ROM or a mix of RAM and ROM
  • Page memory expandable to two display pages
  • Full colour or black and white capability
  • 8 colours for character or background
  • Audio output for tone or white noise

To define a system using the CDP18S661B select 'CDP18S661B Video (NTSC)' from one of the CARD Choice boxes. Note that only one CDP18S661 can be emulated in Emma 02 as such Emma 02 will issue a warning (CARD button will turn red) if two CDP18S661 boards are defined.

To specify the I/O Group, volume, page or character RAM size and/or interrupt mode press the CARD x button and the CDP18S661B Setup window will open. For details see the CDP18S661B manual. Press 'SAVE' to save the configuration.
