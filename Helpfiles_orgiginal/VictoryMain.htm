RCA Studio III / Victory MPT-02 Emulator

This emulator will emulate the RCA Studio III / Victory MPT-02 computer. The Victory is a clone of the RCA Studio III which was never release by RCA. Additional information about the Victory can be found on the Emma 02 site.

To start the RCA Studio III / Victory MPT-02 emulator first select the Studio tab (yellow ellipse) followed by the RCA Studio III / Victory MPT-02 tab (orange ellipse) and then press Start:

The above will have started the default RCA Studio III / Victory MPT-02 emulator configuration without any game cartridge. It will however have loaded the main ROM which includes built in games, try for example to press '3' to start the Bowling game. To change to other predefined configurations load one of them with the menu option below (blue ellipse) following the grey menu:

Summary of the RCA Studio III / Victory MPT-02 configurations:

Chip 8

Studio chip 8 interpreter with one of the chip 8 games selected as default.

Default

RCA Studio III / Victory MPT-02 system ROM, also ready to load a cartridge ROM.

Multi-cart

FLiPs Studio II and RCA Studio III / Victory MPT-02 Multi-cart.
