Software included that uses the VP590 is the game blockout (Blockout [Steve Houk].c8x) and colour kaleidoscope (Color Kaleidoscope [Steve Houk, 1978].c8x) both running on CHIP-8X. Both files can be found in the directory 'Chip-8Chip-8X and Hybrids' (located in the application data directory, see Directory and File Structure)
