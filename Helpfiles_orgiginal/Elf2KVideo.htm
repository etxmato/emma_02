Video Types

To select the specific Cosmac Elf 2000 video type use the choice button 'Video' (yellow ellipse). Currently 2 types are supported: 1861 Pixie graphics and Intel 8275.

1861 Pixie graphics

1861 Pixie graphics is the same graphic system used for the Studio II computer giving a black and white screen of 64 by 128 large pixels.

Intel 8275

The intel 8275 is a video chip for text based output. Different screen sizes can be emulated via software setings of the intel 8275 registers. The Elf 2000 firmware supports this video type but only when a PS2 keyboard is emulated at the same time. Character font ROM is specified and changeable via the 'Intel8275' button (orange ellipse).
