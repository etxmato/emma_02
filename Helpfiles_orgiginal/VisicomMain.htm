Visicom COM-100 Emulator

This emulator will emulate the Visicom COM-100 computer. The Visicom is a clone of the RCA Studio II. Additional information about the Visicom can be found on the Emma 02 site.

To start the Visicom COM-100 emulator first select the Studio tab (yellow ellipse) followed by the Visicom COM-100 tab (orange ellipse) and then press Start:
