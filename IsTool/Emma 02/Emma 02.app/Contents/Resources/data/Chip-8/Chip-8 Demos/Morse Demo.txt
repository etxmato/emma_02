# Morse Code Demo


This demo prints "COSMAC VIP" to the screen while using the sound timer to play

the corresponding Morse code signal.


	morse_demo.ch8 (MM)

Displays "COSMAC VIP" (which means it contains sprites for at least those
characters) while also playing the Morse code for those letters.
