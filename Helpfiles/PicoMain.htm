Pico/Elf V2 Emulator

This emulator will emulate the Pico/Elf V2 computer, which is a self build kit designed by Mike Riley. Additional information about the Pico/Elf can be found on Mikes Pico/Elf V2 page. Pico/Elf V2 is an evolution of the Micro/Elf. Pico/Elf removes the address display, data displays, and input toggles along with associated circuitry in order to add an RS232 port, ROM socket and IDE/CF interface, allowing for a complete machine capable of running Elf/OS.

For details on usage of XMODEM commands xr, xrb, xs, xsb and YMODEM commands yr, ys see Elf chapter: XMODEM Support

To start the Pico/Elf V2 emulator first select the Elf tab (yellow ellipse) followed by the Cosmac Elf, Netronics Elf II or Quest Super Elf tab (orange ellipse):

The default Elf configuration will have been loaded, to change this to a Pico/Elf V2 configuration load one of the configurations with the menu option below (blue ellipse) following the grey menu:

After that press the Start button.

Summary of the Pico/Elf V2 configurations:

Diskless:

Pico/Elf running the diskless ROM

Install Elf/Os 0.3.0:

configuration to install a new Elf/Os version 0.3.0

Install Elf/Os 0.3.2:

configuration to install a new Elf/Os version 0.3.2

Runtime Elf/Os:

Elf/Os running on whatever is installed on the specified .ide file

Runtime Elf/Os - UART:

same as Runtime Elf/Os but using the UART board

STG Runtime Elf/Os:

STG ROM Elf/Os running on whatever is installed on the specified .ide file

STG Runtime Elf/Os - UART:

same as STG Runtime Elf/Os but using the UART board

A basic Elf/Os ide file (pico-installation.ide) is provided in the installation package and when installed (File/Re-Install/1802 Software Files or on initial startup) will be used as default ide file (pico.ide) unless that file already exists.
