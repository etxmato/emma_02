MS2000 Emulator

This emulator will emulate the RCA MicroDisk Development System MS2000. Additional information about the MS2000 can be found on the Emma 02 site.

To start the Microtutor MS2000 emulator first select the RCA tab (yellow ellipse) followed by the MS2000 tab (orange ellipse) and then press Start:

The above will have started the default MS2000 emulator configuration using FDC image files. To change to other predefined conigurations load one of them with the menu option below (blue ellipse) follwoing the grey menu:

Summary of the MS2000 configurations:

FDC

Default confiuration using FDC image files

HD

Configuration using HD Directory simulation with all available MicroDos SW stored in the MS2000/Software folder.
