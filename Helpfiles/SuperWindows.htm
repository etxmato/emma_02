Control Windows

When the 'Elf Windows' (highlighted in yellow) checkbox is selected the Quest Super Elf control window will be shown:

The front panel includes all buttons, LEDs and TIL displays.

Emulation of the TIL displays and the LEDs can take a lot of PC CPU power so default these are only updated every 100 ms. This can be changed via the 'LEDs' field (highlighted in green). When 0 ms (or nothing) is specified the TIL displays and LEDs will be updated in real time.

Two types of TIL displays can be show, TIL 311 or TIL 313, selected via the choice button 'TIL' (highlighted in orange).
