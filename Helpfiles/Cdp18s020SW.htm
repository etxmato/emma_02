Pre SW Load

To run the CDP18S020 Evaluation Kit emulator with SW loaded before the emulator starts select the SW file in the 'RAM SW' field (orange ellipse). To change the location use the 'RAM SW' button (yellow ellipse) to browse for the file, do the same to change the filename or type in a new file name in the text field (orange ellipse).

The selected file will be loaded at RAM address 0 (if the file is binary, other formats like intel hex will be loaded at the specified locations).

The binary, Intel hex and RCS Elf Emulator hex format are supported.

Files can also be loaded in RAM during run time, either by using the Memory Access buttons (see chapter 'Intel Hex and Binary Files' in Memory Access) or via the file area in the terminal section:

When loading a file via the terminal section type the SW filename in the text field (orange ellipse). To change the location use the 'File' button (yellow ellipse) to browse for the folder or file.

Start the emulator and type !M after starting UT4 and then press play (green ellipse). For saving RAM to a file type ?M<start> <length> in UT4 where <start> is the starting address and <length> the number of bytes to be saved. Then press record (red ellipse) in the terminal section and press 'return' in UT4.

To remove the file use the eject button (blue ellipse). During load or save press the stop button (brown ellipse) to stop.
