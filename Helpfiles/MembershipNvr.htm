NVR

Membership Card non-volatile main memory is supported. To turn this feature off, unselect the 'Non-volatile RAM' check box.

To clear main memory at start-up, select the 'Clear RAM' check box. After start-up the emulator will automatically unselect the checkbox.

Notes:

  • For 2 KB to 64 KB RAM configurations: When using non-volatile RAM in combination with loading SW at start-up (see also ROM and RAM Files) first the non-volatile RAM will be restored after which the selected start-up SW will be loaded and overwrite the non-volatile RAM.
  • For 32 KB ROM/RAM configurations: Non-volatile RAM will be restored at the selected RAM location (0 or hex 8000) independently of in what configuration it was saved.
