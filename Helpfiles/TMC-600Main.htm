Telmac TMC-600 Emulator

This emulator will emulate the Telmac TMC-600 home computer. Additional information about the TMC-600 can be found on the Emma 02 site.

To start the Telmac TMC-600 emulator first select the Telmac tab (yellow ellipse) followed by the Telmac TMC-600 tab (orange ellipse) and then press Start:
