ROM Files

To run the COMX emulator at least the COMX basic ROM file should be specified in the 'ROM' field (orange ellipse). Default the COMX ROM location is the Comx directory (located in the application data directory, see Directory and File Structure) and the default filename is 'sb.0000.bin' which is part of the Super Board configuration. To change the ROM file see details in chapter Super Board.

To switch off the Super Board configuration unselect the SB button (green ellipse). After this the default filename is 'comx35.1.3.bin'. To change the location in this configuration use the 'ROM' button (yellow ellipse) to browse for the file, do the same to change the filename or type in a new file name in the text field (orange ellipse):

Similar as for the basic ROM the Expansion Box ROM can be specified via 'EXP ROM' button (blue ellipse) and other interface card ROM files via 'CARD' 1 to 4. The expansion ROM will simulate the COMX expansion box allowing up to 4 interface cards.

The binary, Intel hex and RCS Elf Emulator hex format are supported.
