Directory and File Structure

After initial start-up of Emma 02 a default directory and file structure will be made available according to the sub chapters below. These files can be re-installed at any time with the menu option below (yellow ellipse):

The location for the data directory is default set to:

  • Windows Vista and later: C:\Users\username\AppData\Roaming\Emma 02
  • XP: C:\Documents and Settings\username\Application Data\Emma 02
  • Mac OSX: ~/Emma 02
  • Linux: ~/emma_02_data

Note that when an older versions of Emma 02 was used previously the default data directory might be set to:

  • Mac OSX: ~/Documents/emma_02
  • Linux: ~/Documents/emma_02

To change the location of these files use the menu item below:

A small window will open where with the 'browse' button (yellow ellipse) a new location can be selected. After this 2 options are available: either to only save the new location ('Save' button, orange ellipse) or save and move the existing data directory and files to the new location ('Move & Save' button, blue ellipse). If the save only option is used the data folder should be moved manually. When moving the folder to another disk the process could take a few minutes (depending on the disk speed). In all cases all settings will be set to default as in the ''Set Default (All)' menu action.

Note: When installing a new version of Emma 02 the data directory will NOT be re-installed automatically to avoid overwriting changes made by users. To install all the latest files from the Emma 02 installation package follow the instructions at the top of this page.

Chip 8

Directory including Chip 8, Chip 8X, Chip 10 and Super-chip software. This software can be used in the Cosmac VIP, TMC 2000, Nano, Studio II, Studio III / Victory MPT-02 and where indicated in the ETI660 and Studio IV. Sub directories include:

  • Chip-8 Demos
  • Chip-8 ETI660 Hybrids, SW only running on ETI660 (not compatible with other chip 8 interpreters)
  • Chip-8 Games
  • Chip-8 Hires, SW using an adapted chip-8 interpreter allowing for a higher resolution
  • Chip-8 Hybrids, SW combining 1802 and Chip 8
  • Chip-8 Programs
  • Chip-8X, SW using the chip-8X interpreter
  • Chip-10, SW using the chip-10 interpreter and HW
  • Super-chip Games, SW using my own Super-chip interpreter for a modified VIP (chip10 modification) and Studio IV

Elf/elf-emulation.com

Directory including Elf software available on the Elf/1802 Emulation site. Most software is included with source and help files, stored in sub directories. All software in this directory is copyright by Michael H Riley

To run the following software make sure to select bios1.0.4.rom as main ROM and use the VT100 terminal emulation. Start or auto boot hex address as follows:
'RcForth v1.0.hex': 0
'RcForth Rom Version.hex': E000
'RcLisp Rom Version.hex': D000
'EdtAsm': c000
'EdtAsm Rom Version': 8000
'Bascom Tiny Basic Compiler': 8600
'RcBAsic L1': 8000
'RcBacis L2': 8000

'Dice Rolling.hex' and 'Extended ETOPS' both start at hex 0 and do not need the bios. Note that dice rolling only works on the Cosmac Elf version.
