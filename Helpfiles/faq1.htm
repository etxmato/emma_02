How do I load my SW into an emulated computer?

There are multiple ways to load SW into an emulated computer, choose one of the following options depending on your needs and/or preference:

  1. For loading SW before the computer starts use the ROM, RAM or RAM SW buttons. For more details see the 'ROM Files', 'ROM and RAM Files' or 'pre SW Load' chapters in these help pages for your specific computer.
  2. For loading SW directly into the emulated computer after startup, use the LOAD buttons in the 'Memory Access' area. Depending on what computer is being emulated that will look as in one of the following screenshots:

    For more details see the Memory Access chapter.

  3. For loading SW using the emulated computers cassette interface see the 'Cassette Support' chapter in these help pages for your specific computer.
  4. For loading SW using the emulated computers disk interface see the 'Disk' related chapter in these help pages for your specific computer.
