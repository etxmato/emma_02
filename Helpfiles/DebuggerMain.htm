Debugger

The debugger gives you an inside view of what is going on in the emulated machine. To use the debugger select the 'Tools' tab (yellow ellipse) followed by one of the following (orange ellipse):

  1. 'Debug 1802' to perform tracing, set breakpoints or other use debug commands on the 1802 CPU
  2. 'Debug Pseudo' to perform tracing, set breakpoints or other use debug commands on pseudo code like Chip-8, ST2, ST4, FEL-1, CARDTRAN etc.

Note: none of the settings in the 'Debug 1802' and 'Debug Pseudo' tabs are saved in the Emma 02 configuration, so every time Emma 02 is started these settings are set to default.
