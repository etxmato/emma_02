CDP18S020 Evaluation Kit Emulator

This emulator will emulate the CDP18S020 Evaluation Kit computer. Additional information about the CDP18S020 can be found on the Emma 02 site.

To start the CDP18S020 Evaluation Kit emulator, select the RCA tab (yellow ellipse) followed by the Microboard System tab (highlighted in orange) and then press Start:
