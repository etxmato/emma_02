RCA Studio IV Emulator

This emulator will emulate the RCA Studio IV computer. The Studio IV was never release by RCA. Additional information about the Studio IV can be found on the Emma 02 site.

To start the RCA Studio IV emulator first select the Studio tab (yellow ellipse) followed by the RCA Studio IV tab (orange ellipse) and then press Start:

The above will have started the default RCA Studio IV emulator configuration running Tiny BASIC (PAL & 2020 version). To change to other predefined conigurations load one of them with the menu option below (blue ellipse) follwoing the grey menu:

Summary of the RCA Studio IVconfigurations:

Demo

Studio IV system ROM running the built in color demo.

Super-chip

My own Studio IV SuperChip interpreter! Versions for both PAL and NTSC are included.

Tiny BASIC

3 versions of Tiny BASIC are inlcuded:

  1. The original 1978 version, which was recently found at the Hagley Museum and Library.
  2. NTSC 2020 version ported by me from the VIP version. Half of this is based on the 1978 version (initially we only had half) the other half is based on the VIP code with adaptations to run on the Studio IV.
  3. PAL 2020 version, same as above but adapted to run on a PAL Studio IV.
