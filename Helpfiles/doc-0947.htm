HUG1802/ETI-660 Emulator

This emulator will emulate the HUG1802 and ETI-660 computer. Additional information about these computers can be found on the Emma 02 site.

To start the HUG1802/ETI-660 emulator, select the ETI 660 tab (yellow ellipse) and then press Start (orange ellipse):

After starting the HUG1802/ETI-660 emulator the 'F12' key can be used to simulate the reset button and the 'Insert', '+' or 'numpad +' to simulate the step button. See also Function and Hot Keys.
