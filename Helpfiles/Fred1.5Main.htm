FRED 1.5 Emulator

This emulator will emulate the FRED 1.5 computer. The FRED (Flexible Recreational and Educational Device) concept was used by RCA for multiple prototype computers eventually resulting in the release of computers like the VIP and Studio II. The FRED 1.5 emulated by Emma 02 is based on the information found in this manual as well as Paul Robson's FRED emulator.

To start the FRED 1.5 emulator first select the RCA tab (yellow ellipse) followed by the FRED 1.5 tab (orange ellipse) and then press Start:

The above will have started the default FED 1.5 emulator configuration which is running the Curses Foiled Again game. To change to other predefined configurations load one of them with the menu option below (blue ellipse) following the grey menu:

Summary of the FRED 1.5 configurations:

Default

Set-up to run the Curses Foiled Again game, to select other games use the RAM button or RAM drop down menu.

Fred Demo

Set-up to run the Fred Demo using the voice from the Fred Demo.wav file.

Load

Set-up to load wav files into memory.

The FRED 1.5 is very similar to the FRED 1 but is running on the CDP 1801 instead of the SYSTEM 00 chip set and as such is able to run variants of pseudo code FEL, FPL, GPL, STK and Test-Word, see also Syntax. Additional information about the FRED 1.5 can be found on the Emma 02 site.

The Interlace checkbox will change from the 'dot' screen to full use of all pixels. According to the FRED documentation the dotted screen was used.
