Auto Boot

To enable auto boot select the check box 'Auto Boot' (yellow ellipse). In auto boot mode the CDP18S020 Evaluation Kit will run automatically starting at the boot address 0 or hex 8000 depending on the selected boot type (orange ellipse, program to start at 0, utility to start at hex 8000). To start the CDP18S020 in UT4 select boot type 'Utility'.

When auto boot is disabled the processor will not be started, in this case the processor needs to be started manually with the RUN P (blue ellipse, to start a program at address 0) or RUN U (magenta ellipse, start the utility SW, UT4 at hex 8000) buttons on the front panel.
